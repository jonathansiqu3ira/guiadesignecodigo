# Guia Prático – Design e Código

## Sobre este projeto

O **Guia Prático – Design e Código** é um projeto pessoal e autoral que documenta como eu penso, projeto e implemento interfaces digitais, conectando design, UX e código de forma prática.

Este não é um produto comercial, não é um site institucional e não é um Design System corporativo.  
É um **guia**, no formato de documentação, criado para organizar raciocínio, decisões e boas práticas aplicadas ao dia a dia.

---

## Propósito

Este projeto existe para:

- Registrar decisões de design e implementação
- Conectar UX, UI e front-end de forma realista
- Compartilhar processos e critérios aplicáveis
- Servir como referência para quem atua entre design e engenharia

Tudo aqui é pensado para ser **lido**, **entendido** e **usado**.

---

## Formato do conteúdo

O conteúdo é organizado como uma **documentação técnica**:

- Navegação clara e previsível
- Conteúdo focado em leitura
- Estrutura consistente entre páginas
- Organização pensada para consulta recorrente

---

## O que você vai encontrar aqui

- Fundamentos de UX aplicados a interfaces
- Decisões de UI explicadas com contexto
- Componentes comuns (botões, inputs, etc.)
- Estados, acessibilidade e comportamento
- Considerações técnicas de implementação
- Exemplos práticos e comentários de código

Design e código são tratados como partes do mesmo processo de decisão.

---

## Para quem é este guia

- UX/UI Designers que querem ir além do layout
- Design Technologists
- Front-end Developers
- Pessoas envolvidas na construção de interfaces e produtos digitais

## Stack utilizada

Este projeto é construído com uma stack moderna e simples, pensada para clareza e manutenção.

### Front-end
- React
- Next.js
- TypeScript

### Estilo
- Tailwind CSS
- CSS moderno focado em legibilidade

### Design
- Figma ou Penpot como apoio conceitual
